# 🌸 AI Flower Classifier

## 📌 Project Description
This project is an AI-based Flower Classification system developed for the Advanced Visual Programming Lab Assignment #02.  
It uses machine learning / deep learning to identify flower types from input images.

---

## 🎯 Features
- Image-based flower classification
- AI/ML model prediction
- Fast and accurate results
- Simple user interface (CLI / GUI / Web)
- Supports multiple flower categories
